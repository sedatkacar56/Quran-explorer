# Quran Explorer ✦

An interactive Quran verse explorer — browse all **6,236 verses** of the Holy Quran with Arabic text, English translation, and word-by-word hover tooltips.

🌐 **[Live Demo](https://sedatkacar56.github.io/quran-explorer)** 

---

## ✨ Features

- **Random Verse** — Discover a new verse from the entire Quran with one click
- **Arabic Script** — Uthmani-style Arabic text, beautifully rendered with the Amiri typeface
- **Word Hover Tooltips** — Hover over any Arabic word to see its transliteration and English meaning
- **English Translation** — Sahih International translation displayed alongside every verse
- **Surah Info** — Shows the Surah name, number, verse reference, and a thematic summary for all 114 Surahs
- **All 114 Surahs covered** — From Al-Fatihah to An-Nas, every Surah has a handwritten theme description

## 🛠️ Tech Stack

| Layer | Technology |
|---|---|
| Frontend | Vanilla HTML, CSS, JavaScript |
| Arabic Font | [Amiri](https://fonts.google.com/specimen/Amiri) via Google Fonts |
| Quran Text API | [alquran.cloud](https://alquran.cloud/api) |
| Word Data API | [qurancdn.com](https://qurancdn.com) |
| Translation | Sahih International |

> No frameworks. No dependencies. No build step. Just open `index.html` and it works.

---

## 📡 APIs Used

| API | Purpose |
|---|---|
| `api.alquran.cloud/v1/quran/quran-uthmani` | Full Arabic Quran (Uthmani script) |
| `api.alquran.cloud/v1/quran/en.sahih` | Sahih International English translation |
| `api.qurancdn.com/api/qdc/verses/by_key/` | Per-word transliteration & translation |

All APIs are free and publicly available. No API key required.
